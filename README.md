# Airbnb API

REST & GraphQL API of the Airbnb Clone using Django REST Framework and Graphene GraphQL

### API Actions

- [x] List Rooms
- [x] See Room
- [x] Create Account
- [x] Delete Room
- [x] Update Room
- [x] See Profile
- [x] Edit Profile
- [x] Add Rooms
- [x] Add/Remove Fav Rooms
- [x] Login
- [x] JWT Authentication
- [ ] Filter Rooms
- [ ] Search By Coords
- [ ] Add Room to Favourites
